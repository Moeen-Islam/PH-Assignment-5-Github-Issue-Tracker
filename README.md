# ❓ 1. What is the difference between var, let, and const?

### **Answer:**

**The var keyword** is the old way of declaring variables.It is function-scoped and allows both redeclaration and reassignment, which can sometimes cause unexpected problems.

**The let keyword** is block-scoped, meaning it only works inside the block { } where it is declared. It allows reassignment but does not allow redeclaration in the same scope.

**The const keyword** is also block-scoped, but its value cannot be reassigned after it is declared. But if const is used with objects or arrays, their internal values can still be changed.

# ❓ 2. What is the spread operator (...)?

### **Answer:**

**The spread operator (...)** is used to expand or spread elements of an array or object into individual elements. It helps make code shorter and easier to work with when copying, merging, or passing values.

### **Answer:**

**map()** is used to transform each element of an array and return a new array. The length of the new array stays the same as the original array.

**filter()** is used to select elements that match a condition and return a new array. The new array may be smaller than the original array.

**forEach()** is used to loop through an array and perform an action on each element, but it does not return a new array.

# ❓ 4. What is an arrow function?

### **Answer:**

**An arrow function** is a shorter and simpler way to write a function using the => (arrow) syntax. It was introduced in ES6 to make code cleaner and easier to read.

---

# ❓ 5. What are template literals?

### **Answer:**

**Template literals** are a way to create strings using backticks ( ` ` ) instead of quotes. They allow you to insert variables directly inside a string and also make it easier to write multi-line text.
